# 學校鬧鐘系統 (Python Desktop Edition)

這是一個使用 Python、CustomTkinter 和 Pygame 開發的桌面版學校鬧鐘應用程式。

## 功能概覽

- **鬧鐘排程**: 根據預設的上下課時間表，定時觸發事件。
- **鬧鐘控制**:
    - **獨立控制**: 每個上課或下課鬧鐘都可以獨立啟用或停用。
    - **批次控制**: 提供「全部啟用/停用」、「啟用所有上課/下課鈴」的快捷按鈕。
- **鈴聲管理**:
    - **多元來源**: 支援從「網路 URL」或「本機音訊檔案」新增鈴聲。
    - **播放清單**: 上課與下課鈴聲各自擁有獨立的可管理播放清單。
    - **播放模式**: 每個播放清單都支援「順序播放」與「隨機播放」模式。
- **設定保存**: 所有設定（鬧鐘開關、播放清單、播放模式）都會自動儲存於 `config/settings.json` 中，程式下次啟動時會自動載入。
- **即時狀態**: UI 介面會顯示目前的系統時間與下一個即將響鈴的鬧鐘資訊。
- **測試工具**: 提供「立即測試下一個鬧鐘」和「手動停止鈴聲」的功能，方便使用者驗證設定。

## 如何使用

直接執行 `dist/SchoolBell.exe` 即可啟動程式。

## 如何重新打包

如果修改了原始碼，需要重新打包成 `.exe` 檔案，請依照以下步驟：

1.  確保已安裝 Python 3.12+。
2.  安裝必要的套件：
    ```bash
    pip install customtkinter==5.2.2 pygame==2.6.0 schedule==1.2.0 requests==2.32.5 pyinstaller==6.8
    ```
3.  在專案根目錄 (`bellring`) 下執行以下指令：
    ```bash
    pyinstaller --onefile --windowed --name SchoolBell pybell/main.py
    ```
4.  新的 `.exe` 檔案會生成在 `dist` 目錄下。
